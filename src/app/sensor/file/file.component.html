Sensor Insights(File)

Use this page to generate the min, max, median, and average graphs of a sensor JSON file grouped by its room and day.

Instructions:
  1. Click on the "Open Sensor File" button to select the sensor data you want to analyze.
  2. Hover to the nodes in the chart to see it's value.
  3. Click on the "Save Insights" button to save the grouped data.
